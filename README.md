A Flutter application to explore Dragon Ball characters and planets using the Dragon Ball REST API.
This app allows fans to browse characters and locations from the Dragon Ball universe in a clean and interactive way.

- Built with Flutter
- Browse Dragon Ball Characters
- Explore Dragon Ball Planets
- REST API integration
- Fast and responsive UI
- Material Design based interface

GETTING STARTED

Prerequisites
- Flutter SDK installed
- Android Studio or VS Code
- Emulator or physical device

Installation
1. Clone the repository
   git clone https://github.com/your-username/dragonball.git

2. Navigate to the project directory
   cd dragonball

3. Install dependencies
   flutter pub get

4. Run the app
   flutter run
